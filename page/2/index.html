Linux常用命令总结-3、查看命令历史

一、简介

 1、说明：

   history是Shell的内置命令，用来查看以前执行过的命令。

 2、配置文件

   当登陆Shell时，会读取命令历史文件~/.bash_history中记录下的命令。登录进shell后新执行的命令只会记录在缓存中，这些命令在用户退出时会“追加”至命令历史文件~/.bash_history中。

二、相关参数

history:
        -c: 清空历史命令；  
        -d: 删除历史中的第N个命令；  
         N: 显示最近的N条历史；  
        -a: 追加本次会话新执行的命令至命令历史文件中;  
        -n: 读命令历史文件中未读过的行到历史列表；  
        -r: 读命令历史文件附加到历史列表；  
        -w: 保存历史列表到指定的命令历史文件中；  
        -p: 展开历史参数成多行，但不存在历史列表中；  
        -s: 展开历史参数成一行，附加在历史列表后；

三、history的常用快捷方式

 1. 重复前一个命令的4个方法：
   按↑键并回车执行;
   按 !!并执行回车；
   输入!-1并执行回车;
   按Ctrl+P并执行回车；
 2. 在命令历史中搜索命令：
   Ctrl + r : 在命令历史中搜索命令；
   Ctrl + g : 从命令历史搜索模式中退出；
 3. 要重新调用前一个命令中最后一个参数：
Alt + .:    按住Alt键的同时点击 . 键
Esc, . :    点击Esc键后松开，然后点击 . 键
     !$:    表示
 4. 其他命令历史快捷方式
                  !n:    执行history命令输出对应序列号n的命令；  
                 !-n:    执行history历史中倒数第n个命令；  
                 !:0:    执行前一条命令(去除参数);  
            Ctrl + n:    显示当前历史中的下一条命令，但不执行；  
            Ctrl + j:    执行当前命令；  
             !string:    重复前一个以“string”开头的命令；  
            !?string:    重复前一个包含string的命令；  
           !string:p:    仅打印历史命令而不执行；  
             ^string:    删除上一条命令中第一个string；  
    ^string1^string2:    将上一条命令中的第一个string1替换为string2；  
!:gs/string1/string2:    将上一条命令中所有的string1都替换为string2;
 5. 调用历史参数：
command !^: 利用上一个命令的第一个参数做cmd的参数;  
command !$: 利用上一个命令的最后一个参数做cmd的参数;  
command !*: 利用上一个命令的全部参数做cmd的参数;  
command !:n: 利用上一个命令的第n个参数做cmd的参数;  
command !n:^: 调用第n条命令的第一个参数;  
command !n:$: 调用第n条命令的最后一个参数;  
command !n:m: 调用第n条命令的第m个参数;  
command !n:*: 调用第n条命令的所有参数;  
command !string:^: 从命令历史中搜索以 string 开头的命令，并获取它的第一个参数;  
command !string:$: 从命令历史中搜索以 string 开头的命令,并获取它的最后一个参数;  
command !string:n: 从命令历史中搜索以 string 开头的命令，并获取它的第n个参数;  
command !string:*: 从命令历史中搜索以 string 开头的命令，并获取它的所有参数;
四、命令历史相关环节变量：
HISTSIZE：命令历史记录的条数;  
HISTFILE：指定历史文件，默认为~/.bash_history;  
HISTFILESIZE：命令历史文件记录历史的条数;  
HISTTIMEFORMAT="%F %T"：显示时间;  
HISTIGNORE="str1:str2*:…"：忽略str1命令， str2开头的历史;
五、控制命令历史的记录方式：

 通过修改环境变量：HISTCONTROL的相关参数来控制历史命令的记录方式：

   HISTCONTROL相关参数：

ignoredups:默认，忽略重复的命令，连续且相同为“重复”  
ignorespace:忽略所有以空白开头的命令  
ignoreboth:相当于ignoredups, ignorespace的组合  
erasedups:删除重复命令

   修改方式：

#export HISCONTROL="参数"  
文件存放在 /etc/profile 或 ~/.bash_profile
